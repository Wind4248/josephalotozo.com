# josephalotozo.com

Personal website for Joseph A. Lotozo, CFP® — Edward Jones Financial Advisor, Upper Arlington, Ohio.

## Deploying to GitHub Pages

1. Create a new GitHub repo named `josephalotozo.com` (or any name)
2. Push this folder to the repo
3. Go to repo Settings → Pages → Source: `main` branch, `/ (root)`
4. Point your domain `josephalotozo.com` to GitHub Pages in your domain registrar

## To update content
Edit `index.html` directly — all content and styles are in one file for simplicity.
